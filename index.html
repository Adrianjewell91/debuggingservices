How To Debug and Fix Systems:

  1. 1. Reproduce the Issue.
  2. 2. Gain visibility into the system.
    1. 1. Debugger: View each step of the system in real time.
    2. 2. Elimination: Delete functionality to a working state and then add back in functionality until it breaks.
  3. 3. Refactor and rewrite.
  4. 4. Write tests and document.
